4. Theme

### Set up
#### Clone source then you can run:
```
yarn install
```
#### To install yarn:
https://classic.yarnpkg.com/en/docs/install/#mac-stable
### Launch app:
1. Run on Android device
```
npx react-native run-android
```
2. Run on IOS device
```
npx react-native run-ios
```
